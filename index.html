What is CSS

  1. CSS stands for Cascading Style Sheet

  2. CSS describes how HTML elements are to be displayed on screen,paper,or in other media.

  3. Css is not necccesaryCss saves a lot of work.It can control the layout of multiple web pages all at once

  4. External stylesheets are stored in CSS files.

CSS can be added in HTML document in 3 ways:


  • Inline - by using the style attributes inside HTML elements 1

  • internal - by using a style elements in the head section

  • External _ by using a link elements to link to an external CSS files.

  • The most common way to add CSS,is to keep the styles in external CSS files.However,in this tutorial we will inline and internal styles,because this is easier to demonstrate,and easier for you to try it your self

When comparising CSS class vs ID,the difference is that CSS class applies a style to multiple elements.ID,on the other hand,applies a style to one uniqque elements.ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In,CSS,selection are used to target a specific elements or range of elements on web page.once an elements has been targeted,a style or set of styles can be applied to the elements

There is a wide range of selectors available.Two of the most commonly used are classs and ID.Both are used to target elements to wich a style should be applied.
